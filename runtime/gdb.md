## 关闭其它进程的文件描述符
gdb --pid=<pid>之后call close(<fd>)   
可以使用如下脚本   
```bash
#!/bin/sh
pid=$1
fd=$2
echo "closeing $fd of $pid"
gdb --quiet --pid=$pid -batch -ex 'call close($fd)'
```
